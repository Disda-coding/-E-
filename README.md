# yeb

## 项目目录
### views
里面存放页面
### components
里面存放可复用的组件
### utils
工具类
- api.js
> 封装了axios的接口，实现了响应拦截器
### router
存放页面路由的信息
### assets
一些静态资源文件
### vue.config.js
配置nodejs的代理转发
